# Improve_Oneself

## サイト概要
トレーニングの記録、体重、BMI管理、データやメニューなどを他のユーザーとTwitterのようにつぶやくことで共有しながら楽しく続けられるように。

### サイトテーマ
身体作りやダイエットメニューなどを他のユーザーと共有しながら楽しくトレーニング

### テーマを選んだ理由
なかなか自宅でやる筋トレが続かなかったり、コロナの影響でジムになど通えない人が多いため

### ターゲットユーザ
運動不足やジムに通ったことのない方、トレーニングが一人ではなかなか続かない方向け

### 主な利用シーン
トレーニング時
